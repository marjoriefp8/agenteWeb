# Agente Web con IA

Agente inteligente que combina Claude AI con búsqueda en internet en tiempo real para responder preguntas actualizadas.

## Características
- Búsqueda web en tiempo real usando Serper API
- Claude decide automáticamente cuándo necesita buscar información
- Cita las fuentes de la información encontrada
- Interfaz web con Flask

## Tecnologías
- Python
- Anthropic Claude API (tool use / agentes)
- Serper API (búsqueda web)
- Flask
